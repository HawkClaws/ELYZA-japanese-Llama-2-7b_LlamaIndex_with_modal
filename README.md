# ELYZA-japanese-Llama-2-7b_LlamaIndex_with_modal
modal.comを使って、modalを使ってELYZA-japanese-Llama-2-7b で LlamaIndex を 実行します


[modal](https://modal.com)の登録をおこない、pip install modalしたあとシークレットの設定などを行ってください


## 実行方法

`modal run app.py`
